# Bitcoin Market Sentiment vs Trader Performance Analysis

## Objective

This project analyzes the relationship between Bitcoin market sentiment and trader performance using the Fear & Greed Index and Hyperliquid historical trading data.

## Datasets

### Fear & Greed Index
- Date
- Classification
- Value

### Hyperliquid Historical Trader Data
- Account
- Coin
- Closed PnL
- Size USD
- Direction
- Timestamp

## Methodology

1. Data Cleaning
2. Date Conversion
3. Dataset Merging
4. Exploratory Data Analysis
5. Sentiment-Based Performance Analysis
6. Visualization and Insights

## Key Findings

- Extreme Greed produced the highest average profit per trade (67.89).
- Fear generated the highest cumulative profit (3.36M).
- Extreme Greed achieved the highest win rate (46.49%).
- Traders used the largest position sizes during Fear periods.
